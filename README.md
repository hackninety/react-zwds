# react-zwds · 紫微斗数玄机排盘

文墨天机 APP 风格的紫微斗数 Web 排盘。**算法引擎使用 [iztro](https://github.com/SylarLong/iztro)**（星耀安放、四化、运限全部由它计算），**盘面 UI 完全自研**，不依赖 react-iztro 组件——深空霓虹主题、流光描边、宫位辉光、三方四正流动连线。

## 特性

- **十二宫 + 中宫**：主星/辅星/杂耀竖排带亮度，生年四化实心徽章、运限四化按限色描边徽章；长生/博士/岁前/将前十二神、大限区间、宫干支一应俱全（看流年时岁前/将前自动切流年位）
- **文墨天机式运限拨盘**（底部吸底面板）：`大限 → 流年 → 流月 → 流日 → 流时` 五行联动
  - 点大限（含童限）→ 流年行刷新为该运十年（干支+虚岁）
  - 点流年 → 流月行按**五虎遁**给出各月干支
  - 点流日 → 流时行按**五鼠遁**给出各时辰干支（日柱干支来自 lunar-lite）
  - 行首标签 / 中宫 `本 限 年 月 日 时` 按钮开关各层级盘面显示，`今` 一键回到当前时刻
- **盘面联动**：各宫显示 `大官 / 年子 / 小田 / 月父 / 日疾 / 时兄` 运限宫名徽章与运昌流曲等流耀；**自动选中当前最深运限层的命宫**（看流年即选流年命宫，全关则本命命宫）并点亮**三方四正**发光虚线（金=对照、青=三合），点其他宫可临时切换
- 中宫展示：命造/五行局/起运、阳历农历、时辰、真太阳时、生肖星座、命主身主、命宫身宫、来因宫、四柱干支、当前观测点（含虚岁）；宫内标记**身宫/来因宫**、小限岁数
- **真太阳时**：勾选后展开出生时刻 + 出生地区（省/市/区三级联动，全国区县经度表），按经度差 + 均时差校正排盘（输入按东八区钟表时间解释），中宫同时显示钟表时间与校正结果
- **自化**：宫干四化入本宫（离心）以虚线徽章标注在星耀下方，四化表与运限四化同源（跟随流派设置），并写入 AI 导出
- **AI 导出**：一键导出 JSON / Markdown 全量命盘（十二宫全部星耀神煞+自化、当前五级运限四化与飞宫映射、十二大限总览、当年十二流月总览、流派/盘型口径元信息、可选**常居住地**背景），上传给 AI 即可推理
- **常居住地**（可选输入）：不参与排盘，仅写入导出文件，供 AI 结合迁移宫、方位喜忌、异地发展等议题分析
- **人生K线（分域 + 进出双动能）**：十二宫**各一条**流年运势 K 线，页签切换（综合·命宫 / 事业·官禄 / 财帛 / 感情·夫妻 / 健康·疾厄 / …）。每域按**「一宫看三方四正」**独立评分（本宫×1.0 + 对宫×0.6 + 三合×0.4），命-财帛-官禄同三合会同步、夫妻/疾厄等分化。
  - **区分"进"与"出"两股动能**，解决"赚 100 万花 80 万到底算好算坏"的失真：**进**=禄权科·六合（撑上影）、**出**=忌·冲·自化漏（压下影），**净=进−出** 决定 K 线涨跌。于是「大进大出」年呈**小实体长上下影（十字）**，一眼看出是得失起伏而非单纯好坏；"顺遂"=长阳短影，"破耗"=阴线长下影。
  - **出项分性质**：主动·自化漏（自己花用/投资，自化忌）、被动·忌冲（官非·罚款·外来破财）、纠缠·忌入（是非缠身），悬停即见。
  - 红涨绿跌、金线 5 年均线、大限分段、「今」与选中年标记、身宫域标「身」；**点击年份联动流年拨盘**；全域量化并入 JSON（逐年 进/出/净/形态 + 各域高光/低谷/动能最大年带性质）与 MD（十二域概览含形态 + 综合当前年进出净明细）导出，供 AI 据"净+进出+性质"综合研判
- **流派设置**：安星流派 通行版（南派）/ 中州派，切换时**年界自动套用该派岁首**——南派三合（《全书》体系）默认**正月初一**，中州派（王亭之）默认**立春**，均可手动覆盖；中州派下另可选**盘型**（天盘 / 地盘·身宫起局重排 / 人盘·福德宫起局重排），中州特有杂耀（龙德、劫杀、年解、岁破等）与按年支起命主均由引擎自动切换（iztro 原生口径，保持精简以利 AI 分析）
- **移动端适配**：≤1024px 自动压缩为文墨天机 APP 式高密度整屏盘面，无需横向滚动
- 出生信息支持阳历/农历（含闰月）、早晚子时（timeIndex 0~12），本地记忆上次输入

## 开发

```bash
npm install
npm run dev      # http://localhost:5199
npm run build    # 产物在 dist/
```

## 技术栈

Vite 5 · React 18 · TypeScript · iztro 2.x · lunar-lite

## 目录

```
src/
  core/
    utils.ts       # 干支/五虎遁/五鼠遁/宫名缩写等
    lunar.ts       # lunar-lite 封装（农历⇄公历、日柱干支）
    useZwds.ts     # 排盘主 Hook：拨盘状态 → 目标日期 → horoscope
  components/
    Chart.tsx      # 4×4 盘面 + 三方四正 SVG 连线
    Palace.tsx     # 宫位卡片
    StarCell.tsx   # 星耀 + 四化徽章
    CenterPanel.tsx# 中宫
    HoroscopeBar.tsx # 大限/流年/流月/流日/流时拨盘
    InputPanel.tsx # 出生信息输入
```
