# react-zwds · 紫微斗数玄机排盘

文墨天机 APP 风格的紫微斗数 Web 排盘。**算法引擎使用 [iztro](https://github.com/SylarLong/iztro)**（星耀安放、四化、运限全部由它计算），**盘面 UI 完全自研**，不依赖 react-iztro 组件——深空霓虹主题、流光描边、宫位辉光、三方四正流动连线。

## 特性

- **十二宫 + 中宫**：主星/辅星/杂耀竖排带亮度，生年四化实心徽章、运限四化按限色描边徽章；长生/博士/岁前/将前十二神、大限区间、宫干支一应俱全（看流年时岁前/将前自动切流年位）
- **文墨天机式运限拨盘**（底部吸底面板）：`大限 → 流年 → 流月 → 流日 → 流时` 五行联动
  - 点大限（含童限）→ 流年行刷新为该运十年（干支+虚岁）
  - 点流年 → 流月行按**五虎遁**给出各月干支
  - 点流日 → 流时行按**五鼠遁**给出各时辰干支（日柱干支来自 lunar-lite）
  - 行首标签 / 中宫 `本 限 年 月 日 时` 按钮开关各层级盘面显示，`今` 一键回到当前时刻
- **盘面联动**：各宫显示 `大官 / 年子 / 小田 / 月父 / 日疾 / 时兄` 运限宫名徽章与运昌流曲等流耀；**自动选中当前最深运限层的命宫**（看流年即选流年命宫，全关则本命命宫）并点亮**三方四正**发光虚线（金=对照、青=三合），点其他宫可临时切换
- **飞宫模式**：中宫「飞」开关——点任一宫画出该宫**宫干四化飞线**（禄绿·权蓝·科金·忌紫实线加粗），同目标多化平行错开、自化环形虚圈标注；选中宫右上「详」打开**宫位详情弹层**（三方四正快照/借星、会吉会煞、宫干四化与向心自化、相关格局含赋文、夹宫）
- 中宫展示：命造/五行局/起运、阳历农历、时辰、真太阳时、生肖星座、命主身主、命宫身宫、来因宫、四柱干支、当前观测点（含虚岁）；宫内标记**身宫/来因宫**、小限岁数
- **真太阳时**：勾选后展开出生时刻 + 出生地——**中国**（省/市/区三级联动，全国区县经度表，钟表按东八区解释）或**海外**（IANA 全球时区列表 ~400 区，默认取浏览器系统时区；主城经度取 tzdb 官方坐标，钟表基准按出生日期实算、**自动涵盖历史夏令时**，如东京 +19 分、纽约冬夏 -5/-4、中国大陆 1986~91 夏令时亦被正确处理），按经度差 + 均时差校正排盘，中宫同时显示钟表时间与校正结果
- **自化**：宫干四化入本宫（离心）以虚线徽章标注在星耀下方，四化表与运限四化同源（跟随流派设置），并写入 AI 导出
- **结构分析层（AI 推理增强核心）**：把斗数推理中最机械、AI 徒手最易出错的中间步骤在导出侧确定性算好——
  - **格局检测**：~45 个经典格局（紫府同宫/君臣庆会/机月同梁/阳梁昌禄/火贪铃贪/羊陀夹忌/刑忌夹印/禄马交驰/三奇加会/明珠出海/马头带箭/明禄暗禄/权禄巡逢/文星拱命/擎羊入庙/天乙拱命/紫府夹命/雄宿朝元/寿星入庙/巨逢四煞/科权对拱……），含成格瑕疵判定（三方会煞忌打折注明）与古籍赋文出处
  - **运限格局扫描**：以当前**大限/流年命宫三方**为中心的运限层格局（三奇加会/双禄交会/阳梁昌禄考运/禄马交驰/羊陀夹忌/忌入·忌冲运限命/杀破狼运/火贪引动），直接回答「这一运是不是考运年·财运年·动荡年」，并入 MD 第六节与 JSON/TOON 的 horoscopePatterns 字段
  - **三方四正快照**：十二宫各一行汇总本宫/对宫/三合主星与会吉、会煞、生年四化会入，AI 无需再数宫位
  - **飞宫四化全矩阵**：十二宫宫干四化互飞（12×4）语句化 + **离心/向心自化**汇总
  - **夹宫关系**（左右/昌曲/魁钺/日月/羊陀/火铃/空劫/禄忌夹）与**空宫借星**标注
- **公版赋文库（L3）**：[docs/kb](docs/kb/README.md) 收录二十篇紫微古籍赋文繁体原文（星垣论/骨髓赋/紫微诸星庙陷诀/准绳赋/赋性论/十喻歌/女命赋/太微赋总括/周太常斗临经合婚法/玄微论/玉蟾发微论/百字千金诀/形性赋/太虚观微赋/**诸星问答论**（逐星星情问答 37 节）/增补太微赋/重补斗数彀率/定富贵贫贱十等论含四局/十二宫诸星得地合格诀/失陷破格诀，明清传本公有领域）——AI 深度解读时与导出文件一并选择性上传，格局赋文（L2）与规则速查（L1）以此为根
- **AI 导出**：「**复制给 AI**」一键复制完整分析载荷——**TOON 紧凑数据 + 推理指引 + 规则/星情/主题知识附录**（[TOON](https://github.com/toon-format/toon) 较 JSON 省约 70% token）；另可下载 TOON / Markdown 全量命盘（十二宫星耀神煞+自化、结构分析层、当前五级运限四化与飞宫映射、十二大限总览、当年流月总览含闰月、流派/四化表/子时界口径元信息、可选**常居住地**背景），并内置 **AI 推理指引**（可复制的分析指令模板+提问示例）与**附录A 推理规则速查**（推理次序/四化层级与引动/自化/星情/宫职叠宫/论断分寸的 checklist，公版典籍口径提炼）、**附录B 本盘实际生效四化全表**
- **常居住地**（可选输入）：不参与排盘，仅写入导出文件，供 AI 结合迁移宫、方位喜忌、异地发展等议题分析
- **人生K线（分域 + 进出双动能）**：十二宫**各一条**流年运势 K 线，页签切换（综合·命宫 / 事业·官禄 / 财帛 / 感情·夫妻 / 健康·疾厄 / …）。每域按**「一宫看三方四正」**独立评分（本宫×1.0 + 对宫×0.6 + 三合×0.4），命-财帛-官禄同三合会同步、夫妻/疾厄等分化。
  - **区分"进"与"出"两股动能**，解决"赚 100 万花 80 万到底算好算坏"的失真：**进**=禄权科·六合（撑上影）、**出**=忌·冲·自化漏（压下影），**净=进−出** 决定 K 线涨跌。于是「大进大出」年呈**小实体长上下影（十字）**，一眼看出是得失起伏而非单纯好坏；"顺遂"=长阳短影，"破耗"=阴线长下影。
  - **出项分性质**：主动·自化漏（自己花用/投资，自化忌）、被动·忌冲（官非·罚款·外来破财）、纠缠·忌入（是非缠身），悬停即见。
  - **引动增强**：忌落对宫按「冲本宫」专用权重加重（冲比坐烈）；大限忌+流年忌同引=**双忌叠加**非线性放大；**流曜十颗**（魁钺昌曲禄羊陀马鸾喜，iztro `getHoroscopeStar` 同源公式，鸾喜在夫妻/子女域加倍）逐年参与；流马会禄=**禄马交驰**年；流禄落生年忌宫=**禄忌交缠**变动年；**小限**逐年落宫（生年支三合起、男顺女逆，与 iztro 各宫小限岁数交叉验证一致）入本宫/冲本宫小幅引动；离心自化禄权科忌分级泄气入基调（页签下注明）。
  - **月K线下钻**：「月线」开关（默认开）——选中年份展开逐月细化 K 线（闰年 13 根含闰月位），以该年年K开盘→收盘轨迹为基准，月干四化（五虎遁）+ 月支冲合 + **月曜十颗**制造波动，悬停看逐月引动因子，**点月直接联动流月拨盘**；命宫域逐月表并入 MD 导出、十二域逐月并入 JSON/TOON 导出，AI 定应期月份有据可查。
  - **叠象与结构细化**：**同星叠象**（生年禄星再化禄=叠禄、生年忌星再化忌=忌上加忌×1.3、运限忌撞生年禄=禄逢冲破）；**杂曜按域加权**入基调（鸾喜姚池→婚恋、天巫三台恩光→升迁、孤寡→情感减、天月阴煞→疾厄减、天空破碎→财帛减等，页签下注明）；**交限之年**（踏入新大限首年）动能增；童限年注明仅流年论；**蓝阶背景线**=各大限十年均值，一眼看十年整体抬升/下沉；段标签贪心双行错排防碰撞（童限等窄段自动抬行）。
  - 红涨绿跌、金线 5 年均线、大限分段、「今」与选中年标记、身宫域标「身」；**点击年份联动流年拨盘**；全域量化并入 JSON（逐年 进/出/净/形态 + 各域高光/低谷/动能最大年带性质）与 MD（十二域概览含形态 + 综合当前年进出净明细）导出，供 AI 据"净+进出+性质"综合研判
- **十年规划表**：一限一行的人生总览——大限命宫叠本命何宫（十年主题）、大限四化、命宫域 K 线**均值/高光年/低谷年**、该限**运限格局扫描**，当前限高亮、**点行联动拨盘**；并入 MD 第七节与 JSON/TOON 的 decadePlan 字段
- **十二域雷达图**：K 线区「雷达」开关——选中年份的十二域分数雷达（金点=最强域、紫点=最弱域），与月线并排，一眼看"这年财旺情弱"
- **合盘（双人相性）**：顶栏「合盘」输入乙方生辰（同性/异性通用，流派口径沿用主盘），确定性量化**姻缘·婚恋 / 事业·合伙（正财）/ 金钱·财路（合伙搞钱）**三维评分——年支/命宫支合冲刑害、五行局生克、**太岁入卦**（对方年支坐我何宫：落夫妻=正缘位、落兄弟=合伙位、落财帛=财缘位）、**生年四化互飞**（对方年干禄权科忌打入我盘，双向标注 A→B/B→A）、红鸾天喜对照；三张评分卡+关系概览+规则生成总评（双向化忌交缠、财缘强于情缘等）+互动明细逐条注明依据，一键**复制合盘报告**（MD）喂 AI 深入解读；**流年合参**——随主盘拨盘年份联动，对比双人当年的流年命宫叠宫、四化落宫、命宫域 K 线强弱、太岁（值/冲/刑/害/合）与运限格局，规则生成结论（谁旺谁弱宜谁主导、冲太岁警示、流年忌各落谁的何宫）
- **流派设置**：安星流派 通行版（南派）/ 中州派，切换时**年界与四化表自动套用该派默认**——南派默认**正月初一+通行四化**，中州派（王亭之）默认**立春+中州四化（庚阳武府同·壬梁紫府武，天府化科）**，均可手动覆盖；**四化表整表注入引擎**，生年/运限/飞宫/自化/K线全部同源，实际生效全表随导出注明；另有**子时界**选项（晚子时归次日/当日）；中州派下另可选**盘型**（天盘 / 地盘·身宫起局重排 / 人盘·福德宫起局重排），中州特有杂耀与按年支起命主由引擎自动切换
- **多盘档案册**：输入条「档案」下拉保存常用命盘（按姓名索引、同名覆盖、上限 50），**选择即起盘**；合盘乙方可直接**从档案取人**（真太阳时档案自动折算为校正后的公历+时辰，口径一致）
- **生时校正助手**：时辰不详时点「校时」——同日**十三时辰并排定盘**（命宫/主星/身宫/五行局差异一览），勾选**性格特征**（十项，按命理判定逐盘核对）+ 填入**已发生大事年份**（K线反查该年形态），按匹配分排序推荐；选定时辰一键起盘（自动关闭真太阳时）
- **夏令时提醒**：出生日期落在中国大陆 1986~1991 夏令时区间时（tzdb 逐日精确判定）自动提示回拨 1 小时或切海外模式 Asia/Shanghai 自动换算——不提醒会错一个时辰
- **闰月全支持**：农历输入为**年/月/日三级下拉**（闰月并入月下拉、日数自动 29/30），运限拨盘流月含**闰月位**（沿用本月干支）、流日/流时可达闰月日期、「今」在闰月期间正确定位、导出流月总览含闰月行（lunar-lite 月天数/闰月 API 已知损坏，已改走底层 lunar-typescript）
- **移动端适配**：≤1024px 自动压缩为文墨天机 APP 式高密度整屏盘面，无需横向滚动
- 出生信息支持阳历/农历（含闰月）、早晚子时（timeIndex 0~12）、真太阳时（农历输入亦先转公历再校正），本地记忆上次输入

## 开发

```bash
npm install
npm run dev      # http://localhost:5199
npm run build    # 产物在 dist/
npm test         # vitest：干支/农历闰月/K线引擎/结构分析 27+ 项
```
