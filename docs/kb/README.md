# 紫微斗數公版賦文庫（L3 知識層）

二十篇紫微斗數傳世古籍賦文，**繁體原文照錄**。文本為明清傳本（《紫微斗數全書》系），**早已進入公有領域，無版權問題**；01~14 整理自 ncc.com.tw 紫微古文彙編頁面，15~20 整理自維基文庫「紫微斗數全書／卷一」（zh-hant 轉換），皆僅收古文正文、不含任何現代註解。

## 用途（與 app 知識分層的關係）

- **L1 規則速查**：app 導出 Markdown 的附錄A，是提煉後的推理 checklist；
- **L2 格局賦文**：app 檢測到格局時單條附帶的古籍出處，即引自本庫（骨髓賦為主）；
- **L3 賦文庫（本目錄）**：全文原典。讓 AI 深度解讀命盤時，把相關篇目與導出文件**一併上傳**，供其引經據典、覆核 L1/L2 的口徑。

推薦搭配：論單星星情（最常用）→ 諸星問答論；論性格形貌 → 形性賦；論格局吉凶 → 骨髓賦、太微賦總括；論富貴貧賤定局 → 十等論與四局、得地合格訣／失陷破格訣；論女命 → 女命賦；論廟陷 → 紫微諸星廟陷訣；論合婚 → 周太常斗臨經合婚法（配合 app 合盤報告）。
